# Button
Button component which provides a styled version of button, encapsulating native button inside shadow DOM.
## CSS Variables
|   **Name**  |           **Description**              | **Type** | **Default** | **Optional** |
| :---------: | :-------------------------------------:|:--------:|:-----------:|:-----------:|
| `type`      | `primary`, `secondary`, `hollow`       | text     |   `primary` |   ✔
| `size`      | `small`, `medium`                      | text     |   `small`   |   ✔
| `disabled`  | whether the button should be disabled  | boolean  |    `null`   |   ✔
## CSS Variables
|        **Name**        |   **Default**  |
| :--------------------: |:--------------:|
| `--primary-light`      |    `#66B100`   |
| `--primary-mid`        |    `#3C9700`   |
| `--primary-dark`       |    `#286400`   |
| `--secondary-light`    |    `#FF8800`   |
| `--secondary-mid`      |    `#FF6200`   |
| `--secondary-dark`     |    `#CC4E00`   |
## Slots
|     **Name**   |           **Description**          |
| :------------: | :---------------------------------:|
| ``             | arbitrary content                  |
## Example:
```
<zoo-button disabled="true">
	<div>Disabled :(</div>
</zoo-button>
```
