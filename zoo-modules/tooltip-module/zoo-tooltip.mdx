# Tooltip

Tooltip component.

## Attributes
- position - accepts following values: `top`, `right`, `bottom`, `left`;
  - `optional`
  - `default`: `top`
  - `type` - `text`
- text - text to be displayed inside tooltip;
  - `type` - `text`

## Slots
- `<slot><span>{text}</span></slot>` - arbitrary content which will replace `text` attribute;

## Example usage:
```
<zoo-button>
	<div slot="buttoncontent">
		Button
		<zoo-tooltip text="Tooltip text"></zoo-tooltip>
	</div>
</zoo-button>
```
