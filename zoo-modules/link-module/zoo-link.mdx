# Link

Checkbox component which provides styles for `<input type="checkbox"/>`.

## Attributes
|   **Name**    |           **Description**                                                      | **Type** | **Default** | **Optional** |
| :-----------: | :-----------------------------------------------------------------------------:|:--------:|:-----------:|:-----------:|
| `text`        | what text to present as a link text                                            | text     |             |   ✘
| `href`        | where the link should lead                                                     | link     |             |   ✘
| `target`      | target of the anchor link                                                      | text     |`about:blank`|   ✔
| `type`        | `negative`, `primary`, `grey`, `warning`                                       | text     | `negative`  |   ✔
| `disabled`    | whether the link should be disabled                                            | boolean  |  `false`    |   ✔
| `size`        | `regular`, `bold`, `large`                                                     | text     |  `regular`  |   ✔
## CSS Variables
|        **Name**        |   **Default**  |
| :--------------------: |:--------------:|
| `--primary-light`      |    `#66B100`   |
| `--primary-mid`        |    `#3C9700`   |
| `--primary-dark`       |    `#286400`   |
| `--warning-dark`       |    `#BD161C`   |
## Slots
|      **Name**    |           **Description**          | **Optional** | 
| :--------------: | :---------------------------------:|:-----------:|
| `pre`            | will be rendered before link text, for example: `<svg slot="pre">...</svg>`| ✔
| `post`           | will be rendered after link text, for example: `<svg slot="post">...</svg>`| ✔

## Example usage:
```
<zoo-link href="https://zooplus.github.io/zoo-web-components/" text="Zoo web components" type="primary"></zoo-link>
```
